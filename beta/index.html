SSENSE LINK

TRIGGERS

  • Hover on "SELECT A SIZE" button
  • Hover off "SELECT A SIZE" button
  • Click "SELECT A SIZE" button
  • Hover on any desired size
  • Hover off any desired size
  • Click desired size

RULES

  • Hover on "SELECT A SIZE" button
    • Cursor changes to pointer
    • Button fill fades to black in .5s and stroke is removed from button
  • Hover off "SELECT A SIZE" button
    • Cursor changes back to default
    • Button fill fades to 100% oppacity in .5s and 1px stroke is added to the button
  • Click "SELECT A SIZE" button
    • Select module fades in from below the button in .5s
  • Hover on desired size
    • Cursor changes to pointer
    • Size box will fade to a darker gray in .5s
  • Hover off desired size
    • Cursor changes back to default
    • Size box will fade back to original gray color
  • Click "SELECT A SIZE" button or desired size
    • Size numbers will now appear instead of "SELECT A SIZE"
    • Select module fades out from below the button and returns to original in .5s
  • New size is stored in database that user cannot see
