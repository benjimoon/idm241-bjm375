SSENSE Microinteraction Design

Overview

For my IDM 241 Microinteractions class, the objective was to improve an existing microinteraction on any website of my choice. I programmed and designed a shopping detail page that included 5 triggers for the multi brand retail website, SSENSE. In this class, I built upon my JavaScript, CSS, and HTML skills and put them to use by programming meaningful microinteractions to improve user experience for SSENSE. This project was created during the Fall quarter of 2022, which approximately lasted 3 months.

Background

This was an independent project for my IDM 241 Microinteractions course and I was tasked to improve or add onto a complex microinteraction to improve the website. This project had a timeline of about 3 months where we built an Alpha, a Beta, and a Final build. I used my knowledge of HTML/CSS/JavaScript and applied it to develop more of an understanding on how these skills truly work together in order to create a working prototype.

Problem

Microinteractions are a great way to create a better engagement between the user and the interface. The problem was that SSENSE’s shopping detail page had very poor microinteractions on the page, which led to less user engagement.

Goals

  • Build upon my JavaScript/CSS/HTML skills
  • Learn how to implement 5 meaningful triggers for microinteractions that positively impact the user’s experience
  • Successfully program a working prototype without any errors or issues

Style Guide

For the style guide of my prototype, I wanted it to match SSENSE’s minimalist and contemporary design. I decided to stick to their original color theme of black and white and use “Inter” for the main font.

Developing The Microinteraction

The final microinteraction build was a process of adding onto single microinteractions. There were 3 crucial steps in creating the final product.

Step 1: Alpha Designs

The Alpha Design was the first microinteraction for the project. I first created a wireframe in Figma so I knew exactly what I wanted to develop. After, I developed an “ADD TO BAG” button to the page, which fades from a black background button with white text to a white background button with black text on hover and every hover microinteraction has a hover off which fades back to the original state. I designed this microinteraction after researching what button microinteraction makes a user more likely to click the button. This would allow for more engagement on the website.

Step 2: Beta Designs

For the Beta Design, I added on to my Alpha Design. I also created a wireframe for the beta design in Figma. Above the “ADD TO BAG'' button, I implemented a “SELECT A SIZE” button. When the user hovers over the “SELECT A SIZE” button, the button fades from a black background button with white text to a white background button with black text. When the user clicks on the “SELECT A SIZE” button, a size module fades in from the bottom of the button. When the user hovers over any desired size, the button fades from a black background button with white text to a white background button with black text. Finally, when the user clicks on the desired size, it appears where the “SELECT A SIZE” text used to be and stays there indefinitely.

I created this drop down menu to differentiate SSENSE from any other shopping retail site. Most websites use the default select module for phones, so I decided to add a sleek and engaging microinteraction.

Step 3: Final Designs

For the Final Design, since I already reached the minimum of 5 microinteractions in the Beta Design, I decided to polish the Beta Design. I implemented a black stroke for all the desired sizes so the user can differentiate between the size module and the rest of the website. The only issue I had was making the microinteraction mobile friendly. The hover interaction on mobile had many issues so I created a media query which made the hover interactions only available on the desktop prototype.

This project was an amalgamation of these 3 crucial steps, which helped streamline the overall development of the product.
